# Tienda de Acuarios

## Descripción

Este proyecto es una aplicación web para una tienda de acuarios que permite a los usuarios navegar, seleccionar productos y gestionar un carrito de compras. La aplicación está construida utilizando HTML, CSS y JavaScript, y se basa en Bootstrap para el diseño responsivo.

## Estructura del Proyecto

- **index.html**: Archivo principal que contiene la estructura de la página.
- **CSS/**: Carpeta que contiene los estilos personalizados para la aplicación.
  - **styles.css**: Archivo CSS que define el estilo general de la aplicación.
- **JS/**: Carpeta que contiene la lógica de la aplicación.
  - **app.js**: Archivo JavaScript que gestiona la carga de productos, interacciones del usuario y la funcionalidad del carrito.
- **Media/**: Carpeta que contiene las imágenes de los productos.
- **productos.json**: Archivo JSON que almacena la información de los productos disponibles en la tienda.

## Funcionalidades

- **Carga Dinámica de Productos**: Los productos se cargan desde un archivo JSON y se muestran en la interfaz.
- **Gestión del Carrito**: Los usuarios pueden agregar productos al carrito, ver el contenido del carrito y calcular el total.
- **Interacción del Usuario**: Se utilizan modales para la confirmación de acciones y un formulario de contacto para consultas.
- **Diseño Responsivo**: Utiliza Bootstrap para adaptarse a diferentes tamaños de pantalla.

## Instalación

1. Clona el repositorio o descarga los archivos.
2. Asegúrate de tener un servidor web local (como XAMPP o Live Server) para servir los archivos HTML.
3. Abre `index.html` en tu navegador.

## Uso

1. Navega por los productos disponibles.
2. Selecciona los productos que deseas agregar al carrito.
3. Visualiza el contenido del carrito y el total acumulado.
4. Utiliza el formulario de contacto para realizar consultas.
